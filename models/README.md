# Local Model Files

Place GGUF model files in this directory.

Recommended starting models for this project:

- Qwen2.5 0.5B Instruct GGUF, Q4 quantized
- Qwen2.5 1.5B Instruct GGUF, Q4 quantized
- Llama 3.2 1B Instruct GGUF, Q4 quantized

The model file is intentionally not committed because GGUF files are large.
